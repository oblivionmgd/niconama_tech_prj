# ニコ生コメントで制御するあぷりけーしょんず
ニコ生のコメントで遊べるものをここに置いておきます。  
近いうち、はちゅねミクにコメントでネギを振らせたいと思います。


## 1. niconamaLEDPrj (ニコ生コメントでLチカ)
ニコ生で !G_onとか!B_offとかコメントが打たれると、手元のArduinoに接続されたLEDが光ったり消えたりする。
